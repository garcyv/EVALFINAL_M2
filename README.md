# EVALFINAL_M2
En el contexto de Curso de desarrollo de Aplicaciones Android, se trabajó en el Modulo 2 correspondiente a Base de datos.
Se trabajó con la BD Oracle Express.

Este Módulo tiene una evaluacion final cuyos entregables son los adjunto en este repositorio.
Los Entregables son:
1.- CREATE_INSERT_EVFINAL_M2.sql : SCRIPT con sentencias para la creacion fisica de las tablas e INSERT de datos
2.- CONSULTA_EVFINAL_M2.sql : contine 3 consultas a Base de Datos
3.- DICC_EVALFINAL_M2.PDF: archivo que corresponde al diccionario de datos, creado con Data Modeler
4.- MODELO_EVFINAL_m2.dmd: archivo con imagen del model relacional, creado con Data Modeler
